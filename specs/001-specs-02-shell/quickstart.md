# Quickstart Guide: 02-shell-scripting

Guide to run and solve Module 02.

## How to execute

1. SSH into the VM:
   ```powershell
   vagrant ssh
   ```
2. Navigate to directory:
   ```bash
   cd /labs/02-shell-scripting/
   ```
3. Run the interactive demo:
   ```bash
   ./demo.sh
   ```
4. Solve the challenge in `challenge/file_filter.sh`.
5. Run the validator:
   ```bash
   ./verify.sh
   ```
